# 加禾出货单

一个轻量的网页版出货单工具，专为辅料批发场景设计。手机打开即用，填完一键导出图片或 PDF 发给客户。

🔗 **在线访问**：https://eryiyao.github.io/jiahe/

---

## 为什么做这个

每次发货都要手写一张纸质出货单，再拍照发给客户，重复且容易出错。微信里也找不到一个足够轻、足够顺手、又能直接出图的小工具 —— 现成的发货单 App 都太重，飞书多维表格不擅长做"一张好看的单据"。

于是用一个单文件 HTML 解决问题：浏览器打开，填完导图，发给客户，结束。

## 功能特性

- 📝 **表单填写**：客户、单号、日期、商品明细，实时预览
- 💰 **金额自动计算**：每行小计、总计，并实时转成中文大写（"壹仟贰佰叁拾肆元伍角陆分"）
- 📋 **辅料库**：预设品名 + 单位 + 单价，开单时下拉选择自动填入，避免重复输入和单价记错
- 🖋️ **手写签名**：在签名区直接用手指/鼠标签名，跟随导出的图片一起保存
- 📷 **一键导出**：高清 PNG 图片或 PDF 文件，文件名自动带客户名和日期
- 💾 **本地存储**：店铺信息、辅料库都存在浏览器本地，下次打开自动恢复
- 📲 **移动端友好**：响应式布局，iPhone Safari 可"添加到主屏幕"像 App 一样使用
- 🔄 **辅料库同步**：支持 JSON 文件导入导出，也支持从 Excel / 飞书直接复制粘贴批量导入

## 使用方式

### 网页版（推荐）

1. 用 Safari / Chrome 打开 https://eryiyao.github.io/jiahe/
2. 填写店铺信息（首次使用需要填一次，之后自动记住）
3. 录入客户名、商品明细
4. 点击"导出图片"或"导出 PDF"
5. 把图片发给客户

### 添加到手机主屏幕（iPhone）

1. Safari 打开网址
2. 点底部分享按钮（方框 + 向上箭头）
3. 选择"添加到主屏幕"
4. 桌面会出现一个图标，以后点图标直接进入，全屏运行

### 离线使用 / 自部署

整个项目就是一个 `index.html` 文件，下载到本地双击即可使用，不需要任何服务器或安装步骤。

## 技术说明

- 纯前端单文件，无后端、无构建步骤
- 数据存储在浏览器 localStorage，每个设备独立
- 依赖两个 CDN 库：[html2canvas](https://html2canvas.hertzen.com/) 用于截图导出，[jsPDF](https://github.com/parallax/jsPDF) 用于 PDF 生成

## 关于跨设备数据同步

由于使用浏览器本地存储，店铺信息和辅料库**不会在多设备间自动同步**。如需同步，可使用辅料库的"导出文件 / 导入文件"功能手动操作。
